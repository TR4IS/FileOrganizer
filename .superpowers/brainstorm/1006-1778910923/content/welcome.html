FileOrganizer — Electron Rebuild

We're designing the new version together. I'll show mockups and layout options here as we go.

What we're building

A beautiful Electron desktop app (Windows-first) that watches a folder and automatically organizes files by type — images, videos, PDFs, audio, etc. The current Python/CustomTkinter version works well but is limited in visual polish. The Electron rebuild will have a modern, rich UI.

Current features to carry over

  • Select target folder to organize
  • Manual "Organize Now" button
  • Background file watcher (auto-organize on new files)
  • Activity log view
  • System tray support (minimize to tray)
  • Auto-updater
